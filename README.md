Diabetes Risk Prediction Model (Machine Learning)
I built an end to end machine learning solution to predict diabetes risk from health indicators, using a large real world dataset of over 250,000 patient records.
What I did:

Cleaned and explored the dataset to identify key health risk factors
Built and compared four machine learning models (Logistic Regression, Decision Tree, Random Forest, XGBoost) to find the best performer
Tuned the top model using automated hyperparameter optimization for better accuracy
Evaluated performance using industry standard metrics (accuracy, precision, recall, F1 score, ROC AUC)
Made the model interpretable using SHAP, so predictions can be explained in plain terms (which factors drove a specific risk score, not just a black box output)

Tools used: Python, Pandas, NumPy, Scikit-learn, XGBoost, SHAP, Matplotlib
