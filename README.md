# 🩺 Diabetes Prediction Using Machine Learning

## 📌 Project Overview

This project presents an end-to-end machine learning pipeline for predicting diabetes using health-related indicators. Multiple classification algorithms were developed, compared, and interpreted to identify the most effective model for diabetes prediction.

The workflow includes exploratory data analysis (EDA), data preprocessing, model development, hyperparameter optimization, and explainable AI using SHAP.

---

## 📂 Dataset

- Dataset: Diabetes Health Indicators Dataset
- Samples: 253,680
- Target Variable: Diabetes_binary

---

## 🚀 Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- Optimized XGBoost (RandomizedSearchCV)
